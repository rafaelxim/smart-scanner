## Especificacao

Build the mobile UI for selecting or capturing receipt images and sending them to the backend.
Support both camera and gallery flows.

## Entregavel

- Camera capture action
- Gallery selection action
- Upload button and loading state
- Error state for failed uploads
- Basic success feedback after upload

## Definicao de pronto

- A user can choose an image from the camera or gallery
- The selected image can be uploaded to the backend
- The UI shows loading and failure states clearly

## Teste

- Upload a camera image successfully
- Upload a gallery image successfully
- Trigger a network failure and verify the error state appears
