## Especificacao

Add backend endpoint for current-month dashboard totals.

## Entregavel

- Endpoint for monthly spending summary
- Total spent for selected month
- Recent receipts for selected month
- Month parameter support

## Definicao de pronto

- Endpoint defaults to current month
- Endpoint can return another month by parameter
- Totals are based on confirmed receipts only
